What is C#?

  • C# is an open source object-oriented programming language made by Microsoft built on the .Net Framework.
  • It is still being worked on to this day and is about to release version 9.0. The framework which it runs on is also about to hit version 5.0. This makes it a very mature language.
  • It is very similar to Java with Microsoft making it at about the same time that Java was getting popular, so this makes it a great language to dive into after we learned Java!

Now to take a look at just how similar C# and Java are in syntax.

Features of C#

  • It is modern as modern day

    Because of its ever growing feature set and continous updates and improvements over time, every feature is very up to date. For example, a new official library for cross platform Graphical User Interfaces (GUI) is coming out this year called MAUI!

  • Cross Platform

    Because of .Net Core, the framework C# runs on, you can write and compile its code on any platform and like mentioned before, an offcial cross platform GUI library is coming out with many third party solutions just as good if not better. It is also able to compile to Web Apps, iOS/Android Apps, Windows and Linux and MacOS apps, and with Unity most modern game consoles!

  • Provides Many Solutions

    Like mentioned before, C# can make Native Windows Apps, Windows libraries, Windows Components, Windows Services, Web Apps, Web services and Web API, Native iOS and Android mobile apps, Backend services, Azure cloud applications and services, Backend database using ML/Data tools, Interoperability software such as Office, SharePoint, SQL Server, Artificial Intelligence and Machine learning, Blockchains and distributed ledger technology including cryptocurrency, Internet of Things (IoT) devices, Gaming consoles and gaming systems, Video games and MORE!

  • Very Popular

    According to Stack Overflow, about 30% of Developers claim to use C# regularly, with it having the third biggest Stack Overflow community, also Stack Overflow is built using C# (ASP.NET). There are more than 5,000 jobs for C# advertised each month in jsut the United States. Many people can just pick up and learn C# because many parts of it just work and make sense but can also be a huge learning hole.
